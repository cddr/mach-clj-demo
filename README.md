# mach-clj-demo

Quick demonstration of how to build Clojure projects using
juxt/mach in combination with "deps and the cli"

See setup instructions for mach

### Demo

```bash
git clone https://github.com/cddr/mach-clj-demo.git
cd mach-clj-demo
mach test
mach run
mach uberjar
```

### References:
